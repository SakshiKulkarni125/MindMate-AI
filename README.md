# 🧠 MindMate AI

MindMate AI is a Retrieval-Augmented Generation (RAG) based Mental Health Awareness Assistant built using Streamlit, FAISS, Sentence Transformers, and Groq LLMs.

The application allows users to ask questions related to:
- stress
- anxiety
- burnout
- emotional wellness
- coping strategies
- student mental health

The chatbot retrieves relevant information from mental health awareness PDFs and generates supportive, easy-to-understand responses using an LLM.
